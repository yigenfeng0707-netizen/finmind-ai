- [ ] Which slides feel rushed?
- [ ] Which transitions feel awkward?
- [ ] What's the natural breathing point?

### Afternoon Session (2:00-2:30 PM, 30 min)

**2:00-2:15 (15 min) - Section-by-section practice**

- [ ] **Section 3 (Old Way)**: 15 seconds

- [ ] **Section 10-12 (Architecture + Comparison + Numbers)**: 50 seconds
- [ ] **Section 13-14 (Business + Roadmap)**: 35 seconds
- [ ] **Section 15 (CTA)**: 30 seconds

**2:15-2:30 (15 min) - Demo backup practice**
- [ ] Practice opening Vercel URL in browser (3 times)

### Evening Session (8:00-8:15 PM, 15 min)

**8:00-8:10 (10 min) - Record yourself**
- [ ] Set up phone camera at eye level
- [ ] Record full 5-minute speech (using script, not memorized)

**8:10-8:15 (5 min) - Quick review**
- [ ] Watch the recording
- [ ] Note 3 things to improve tomorrow
- [ ] Save recording as `day1_v1.mp4`

### Day 1 Checklist
- [ ] Read script 3 times
- [ ] Practice each section once
- [ ] Record first version
- [ ] Identify 3 improvement areas

## 📅 DAY 2 (JUNE 9, TUESDAY) - 脱稿练习

**Daily theme:** Speak naturally, hit the time

### Morning Session (9:00-9:45 AM, 45 min)

**9:00-9:15 (15 min) - Cold speech (no script)**
- [ ] Try to deliver the full 5 minutes from memory
- [ ] Use bullet points only (1 page summary, not full script)
- [ ] Don't stop for mistakes - keep going
- [ ] Time yourself

**9:15-9:25 (10 min) - Timing analysis**
- [ ] Were you under 4:30? → Add pauses, slow down
- [ ] Were you over 5:00? → Cut 1 sentence per section
- [ ] Were you at 4:45-5:00? → Perfect, just polish

**9:25-9:45 (20 min) - 2nd and 3rd attempts**
- [ ] Try 2 more times
- [ ] Focus on natural delivery
- [ ] Don't over-rehearse - keep it conversational

### Afternoon Session (2:00-2:45 PM, 45 min)

- [ ] Ask a friend, family member, or colleague
- [ ] Do the full 5 minutes

  - "Where did I seem nervous?"

**2:15-2:30 (15 min) - Q&A practice**
- [ ] Pick 5 questions from `DEMO_SCRIPT_EN.md`
- [ ] Practice answering them out loud
- [ ] Time yourself (aim for 30-60 second answers)

**2:30-2:45 (15 min) - Refinement**
- [ ] Apply feedback from test audience
- [ ] Make small adjustments
- [ ] Don't completely rewrite - just polish

### Evening Session (8:00-8:15 PM, 15 min)

**8:00-8:10 (10 min) - Final record**
- [ ] Record the speech again
- [ ] Compare to Day 1 version
- [ ] Save as `day2_v1.mp4`

**8:10-8:15 (5 min) - Self-eval**
- [ ] Did you improve? (yes/no)
- [ ] What's the #1 thing to fix tomorrow?
- [ ] Are you feeling more confident? (1-10)

### Day 2 Checklist
- [ ] Spoke from memory 3 times
- [ ] Time within 4:45-5:00
- [ ] Got feedback from real person
- [ ] Practiced 5 Q&A answers
- [ ] Recorded version 2

### Day 2 Targets

2. ✅ Speak conversationally, not robotically
3. ✅ Make eye contact
